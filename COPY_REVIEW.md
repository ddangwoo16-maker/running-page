# 런닝머신 거치대 + 전자책 랜딩페이지 — 전체 카피 정리

## 프로젝트 개요
**목표**: 사용자(판매자)가 15개월간 걷기→마라톤 완주까지 경험한 실제 기록을 보여주면서, 그 과정에서 습득한 "실패 예방법"을 전자책으로 판매하는 페이지

**핵심 메시지**: 
- "노력했다" X
- "실패만 피했다" O (부상 예방, 방법론 중심)
- "거창한 계획 없었다"
- "같은 시간, 같은 장소, 꾸준히"

### 📍 **섹션1 — 히어로 (풀스크린)**
**위치**: 페이지 최상단  
**시각적 구성**: 마라톤 레이스 사진 → 텍스트

**카피1-1 (메인 헤드)**
```
하지만, 풀코스 마라톤 기록
4시간 45분 18초
```
**문맥**: 히어로 상단에서 "달성 사실"을 선언  
**왜?**: 결과를 먼저 보여줘서 가능성을 증명. 골드 컬러로 숫자만 강조하는 이유는 수치의 객관성을 부각  
**대체 가능?**: YES - 더 임팩트 있는 문구로 개선 가능

**카피1-2 (서브 헤드)**
```
특별한 계획은 없었습니다. 그냥 실패만 피했을 뿐.
부상을 피하고, 재미있게 했습니다. 그래서 일반인인 저도 했습니다.
```
**문맥**: 성과 다음에 "방법"을 설명  
**왜?**: "노력 강조" 대신 "방법론"을 즉각 제시. "일반인인 저도"는 공감대 형성  
**우려**: "특별한 계획은 없었습니다"는 다소 겸손과 자신감이 섞여 있음. 더 단호할 수 있음  
**개선 아이디어**: "거창한 계획보다는 올바른 순서가 전부였습니다"

### 📍 **섹션2 — 바닥 (2024년 11월 시작 전)**
**위치**: 스크롤 후 첫 섹션  
**시각적 구성**: 라벨 "2024년 11월" → 셀카 사진 → 텍스트

**카피2-1**
```
39세. 야식이 습관이었고, 몸은 정직하게 그 결과를 보여주고 있었습니다.
여기서 시작했습니다. 이 상태에서 부상 없이 어떻게 나올지가 전부였습니다.
```
**문맥**: "시작점" 제시 → 즉시 "방법" (부상 예방)으로 연결  
**왜?**: 
- "야식이 습관" = 공감 유도 (흔한 상황)
- "부상 없이 어떻게 나올지" = 핵심 과제 명시
- 의지/다짐이 아니라 **문제 해결의 프로세스** 제시

**우려**: "여기서 시작했습니다"는 다소 막연함. 더 구체적인 감정이나 상황 제시 필요할 수 있음  
**개선 아이디어**: "이 사진 찍고 석 달을 더 미뤘습니다. 하지만 시작할 땐 한 가지만 생각했습니다: 부상 없이 어떻게 나올까."

**우려**: "두 번째 방법"이라는 카운팅이 직설적. 더 자연스럽게 흐를 수 있음

## 📊 **카피 톤 분석 및 개선 제안**

### 현재 톤의 강점
1. **"실패 예방" 중심** — "노력/의지" 대신 "방법론" 강조 ✓

3. **현실성 제시** — "가기 싫은 날도 있었습니다" ✓
4. **단순함 강조** — "거창한 계획 없었습니다" ✓

### 현재 톤의 약점
1. **반복이 많음** — "특별한 계획은 없었습니다", "거창한" 등 키워드 반복
2. **추상성** — "방법이 있으면", "방법이 작동했습니다" 등이 구체성 부족
3. **감정의 부족** — 너무 건조함. 공감 요소 부족
4. **논리적 연결** — 섹션 간 톤의 부조화 (특히 섹션2→2.5→3)

### 개선 방향
**추천하는 재검토 포인트:**
- "거창한 계획" → "올바른 순서" / "정해진 방법" 등으로 다양화
